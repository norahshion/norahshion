# プロフィール
芝浦工業大学鳥人間部TeamBirdmanTrial(https://github.com/tbt-densou) にて25代電装設計を担当していましたnorahshionです。<br>
ESP32マイコンを用いたデータ取得やBluetooth通信、Androidアプリ（Kotlin/Jetpack Compose）によるリアルタイム航空管制システムを一気通貫で開発してきました。<br>
また、大学では画像センシング研究室に所属して画像処理・機械学習（Python）の研究に励む傍ら、個人アプリ開発にも注力しています。

# 🚀 Featured Projects

### 📊 鳥人間部：航空管制・機体データリアルタイム同期システム
ESP32からBluetooth経由でAndroidアプリへセンサーデータを送信し、Firebaseでクラウド共有するシステム
- **担当:** 設計，メインAndroidアプリ開発、組み込みファームウェア開発、他班との仕様調整
- **技術:** Kotlin, Jetpack Compose, MVVM, ESP32, Firebase
- **成果:** 徹底した非同期処理（Coroutines）の最適化とテストシミュレータの自作により、テストフライトにて**リアルタイム性**を追求。
